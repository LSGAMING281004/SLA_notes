block level elements
----------------------------------
  this elements takes the entire width of the screen irrespective of the content's size

inline elements
-------------------------------
  this element takes the width as much as the content is required


block level elements
1. div
2. p - para
3. h1 - h6

inline elements
----------------------
1. span
2. img
3. a - anchor tag

formatting tags
------------------------------
  it is used to format the contents

here, <strong> </strong>, <em></em>, <ins></ins> outputs looks like <b></b>, <i></i><u></u> tags but the 'strong', 'em', 'ins' are semantic tags also it convey the importance of the data not only the appearance.


semantic tags: tags which conveys its meaning in the tag name itself

examples:
-------------------
<header> </header> tags
<footer> </footer> tags
<img></img> tags
<table> </table> tags

---------------
<html>

<head>


</head>

<body>
